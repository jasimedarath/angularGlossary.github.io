React Context API

Context provides a way to pass data through the component tree without having to pass props down manually at every level. It's designed to share data that can be considered "global" for a tree of React components.

Creating and Using Context

Context lets you pass data deep into the component tree without prop drilling.

  • Create context with createContext()
  • Provide value with Provider component
  • Consume value with useContext hook
  • Avoids prop drilling through intermediate components
  • Value accessible to all nested consumers

Performance Optimization

Context can cause performance issues if not used carefully.

  • Memoize context value to prevent re-renders
  • Split contexts by update frequency
  • Separate state and dispatch contexts
  • Use React.memo on expensive consumers
  • Consider state management libraries for complex apps
