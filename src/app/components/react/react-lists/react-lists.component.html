React Lists & Keys

Learn how to render multiple components using JavaScript's map() function and understand the importance of keys for React's reconciliation algorithm.

Rendering Lists

Use JavaScript's map() function to transform arrays into lists of elements.

  • map() creates a new array of JSX elements
  • Each element needs a unique key prop
  • Keys help React identify changes
  • Common pattern for rendering arrays
{{basicLists}}

Importance of Keys

Keys help React identify which items have changed, been added, or removed.

  • Keys must be unique among siblings
  • Use stable IDs, not array indices
  • Index as key causes issues with dynamic lists
  • Keys improve reconciliation performance
  • Don't use random values as keys
{{keysImportance}}

Nested Lists

Render nested data structures with multiple map() calls.

  • Each level needs unique keys
  • Use map() for each nesting level
  • Keys only need to be unique among siblings
  • Extract nested lists to components for clarity

Use JavaScript array methods to filter and sort data before rendering.
