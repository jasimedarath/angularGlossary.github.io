React Events

React events are named using camelCase and you pass a function as the event handler. React uses synthetic events that wrap native browser events for better cross-browser compatibility.

Basic Event Handling

React event handlers are written in camelCase and passed as functions, not strings.

  • onClick, onChange, onSubmit (camelCase)
  • Pass function reference, not function call
  • Can define handlers inline or separately
  • Use arrow functions to pass parameters
  • Event handlers receive synthetic event object

Synthetic Events

Synthetic events are React's cross-browser wrapper around native browser events.

  • Same interface as native events
  • Works consistently across all browsers
  • Access native event via e.nativeEvent
  • No longer pooled in React 17+
  • Can be used asynchronously
{{syntheticEvents}}

Form Events

Common form events: onChange, onSubmit, onFocus, onBlur.

  • onChange: Fires on every input change
  • onSubmit: Fires on form submission
  • onFocus: Element gains focus
  • onBlur: Element loses focus
  • Use e.preventDefault() to prevent default behavior
{{formEvents}}

Mouse and Keyboard Events

Handle mouse movements, clicks, and keyboard input.

  • Mouse: onClick, onDoubleClick, onMouseMove, onMouseEnter, onMouseLeave
  • Keyboard: onKeyDown, onKeyUp, onKeyPress (deprecated)
  • Access key info: e.key, e.code, e.keyCode
  • Check modifiers: e.ctrlKey, e.shiftKey, e.altKey
  • Get mouse position: e.clientX, e.clientY
{{mouseKeyboardEvents}}

Event Bubbling and Propagation

Events bubble up from child to parent elements by default.

  • Events bubble up the DOM tree
  • Use e.stopPropagation() to stop bubbling
  • e.preventDefault() prevents default browser behavior
  • Can handle at parent level (event delegation)
  • Capture phase available with onClickCapture
{{eventPropagation}}

Custom Event Handlers

Create custom event handlers and pass custom data to parent components.

  • Define custom callback props
  • Process events and pass custom data
  • Use for component communication
  • Debounce frequent events for performance
  • Throttle events like scroll or resize
