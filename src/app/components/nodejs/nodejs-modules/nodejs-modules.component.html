Node.js Modules

Understanding module systems in Node.js - CommonJS and ES Modules.

CommonJS Modules

CommonJS is the traditional module system used in Node.js.

  • require() to import modules
  • module.exports to export
  • Synchronous loading
  • Widely used in Node.js ecosystem
{{commonjsModules}}

ES Modules (ESM)

ES Modules are the modern JavaScript module standard.

  • import/export syntax
  • Static analysis support
  • Tree shaking friendly
  • Async module loading
{{esModules}}

Built-in Node.js Modules

Core modules provide essential functionality without external dependencies.

  • fs - File system operations
  • path - Path manipulation
  • http - HTTP functionality
  • url - URL parsing
{{builtinModules}}

Working with npm Modules

npm packages extend Node.js with third-party functionality.

  • Version control with semver
  • Development vs production dependencies
