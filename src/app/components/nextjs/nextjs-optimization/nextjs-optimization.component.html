Next.js Optimization

Next.js provides built-in optimizations for images, fonts, scripts, and more to improve performance and user experience.

Image Component

next/image automatically optimizes images with lazy loading, responsive sizes, and modern formats.

  • Automatic WebP/AVIF format conversion
  • Responsive image sizing
  • Lazy loading by default
  • Prevents Cumulative Layout Shift (CLS)
  • On-demand optimization
  • Priority loading for LCP images
{{imageOptimization}}

Font Optimization

next/font automatically optimizes and self-hosts fonts with zero layout shift.

  • Automatic self-hosting of Google Fonts
  • Zero layout shift with font-display: swap
  • No external network requests
  • Built-in font subsetting
  • Local font support
  • CSS variable support
{{fontOptimization}}

Script Component

next/script optimizes third-party script loading with custom strategies.

  • Control loading priority
  • beforeInteractive for critical scripts
  • afterInteractive for analytics (default)
  • lazyOnload for non-critical scripts
  • Automatic deduplication
  • Load callbacks
{{scriptOptimization}}
